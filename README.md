# Alpine Resort Showdown 🎿

A visual comparison of five top ski resorts for a late-March 2026 trip:

- **Courchevel** — Three Valleys, France
- **Méribel** — Three Valleys, France
- **Val Thorens** — Three Valleys, France
- **Verbier** — 4 Vallées, Switzerland
- **Chamonix** — Mont Blanc, France

Includes terrain ratings, late-March snow reliability, après-ski scores, and accommodation estimates for 3 people.
